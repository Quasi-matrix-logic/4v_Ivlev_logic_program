There are 9 Ivlev's logics, which are denoted as: Sa, Sc, Sb, Sd, Se, Sf, Sg, Sh, Si. To run the program, you need to press START. The calculation results are displayed in this text field and in the browser console.

The program will show a list of implications between modal patterns. If you select "PRINT FALSE", you will also see false implications. Attention. The program may freeze if you set a large pattern length.

The "Axioms" tab contains a list of all axioms of all Ivlev's logics. Click "CALC" to check their general validity in the selected semantics.


In the "Inclusions" tab, you can check inclusions between Ivlev logic graphs. More precisely, between graph fragments up to the selected pattern length value.
